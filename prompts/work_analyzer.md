# 人物志分析 Prompt

## 任务

你将收到 **{name}** 的原著文本片段（章节正文、番外、对白等）。
从中提取他/她的人物志信息，用于构建 Profile。

**原则：只提取文本中有明确依据的内容，不要推断，没有就标注"原著文本不足"。**

---

## 通用提取维度（所有角色适用）

### 1. 故事定位

从文本中识别：
- 角色在哪些故事线/弧段中出现（正文 / 哪条番外线）
- 在故事中承担什么叙事功能（推动剧情 / 情感锚点 / 对立面 / 工具人）
- 与哪些核心角色构成关键关系（盟友/敌对/羁绊/牵制）
- 原著中其他角色如何描述或评价 ta（直接引用）

```
输出格式：
出场弧段：[列表]
叙事功能：[描述]
核心关系人：[姓名 — 关系类型]
他人评价：["原话", ...]
```

### 2. 能力设定

从文本中提取：
- 明确描述的能力/技能/武力/特殊能力（原话引用）
- 能力的上限与已知边界（极限是什么，在哪场战斗/事件中体现）
- 能力的代价或限制条件（如果有）
- 他/她不擅长的事（文本有提及）

```
输出格式：
核心能力：[列表，附原文出处]
能力上限：[描述]
限制条件：[描述或"原著未提及"]
弱点：[描述或"原著未提及"]
```

### 3. 关键剧情节点

提取角色的关键行动/选择/转折点（按时间顺序）：
- 每个节点：发生了什么 → 角色做了什么选择 → 结果
- 重点关注：揭示性格的选择（而非单纯的能力展示）
- 标注章节范围（如"第XX-XX章"或"某番外"）

```
输出格式：
节点 N（位置）：[事件] → [选择] → [结果]
```

### 4. 关系图谱

对每个与 {name} 有明显互动的角色：
- 关系性质（亲密/对立/依赖/牵制/利用/真心）
- 关系变化轨迹（是否发生过转折）
- 代表性互动场景（引用原文）

```
输出格式：
{关系人名}：[关系性质] — [变化轨迹] — [代表场景引用]
```

---

## 角色类型专项提取

根据 {name} 的故事职能，重点提取对应维度：

---

### ⚔️ 武将 / 战士型

**战斗风格**：
- 擅长的战斗类型（近战/远程/群攻/单挑/战场指挥）
- 代表性战斗场景及结果
- 克制他/她的对手和情况
- 受伤/落败时的应对方式（逃/硬撑/求援/放弃）

**战略思维**：
- 是否有独立的战略判断，还是执行型
- 如何权衡己方损失与目标达成
- 对敌我双方实力的评估习惯

---

### 🧠 谋士 / 智者型

**谋略范围**：
- 擅长的计谋类型（情报/外交/军事/经济/人心）
- 原著中完整呈现的计划（结构描述）
- 最终被谁识破或推翻（如果有）

**信息处理**：
- 如何收集情报（主动/被动/依赖他人）
- 推断时的依据（逻辑/直觉/经验）
- 判断失误时的反应

---

### ✨ 修士 / 异能者型

**修为与异能**：
- 功法体系/异能来源（如原著有世界观设定）
- 当前修为段位/能力等级
- 使用能力的代价或限制
- 能力在关键剧情节点的运用

**成长轨迹**：
- 修为的变化节点（何时突破/何时受限）
- 影响成长的关键事件

---

### 👑 领导者 / 权力型

**权力结构**：
- 实际掌握的权力范围
- 正式权力 vs 实际影响力的差距（如有）
- 权力的来源（出身/军功/谋略/情感控制）

**治理方式**：
- 对下属的管理风格（集权/分权/放任）
- 对盟友/附庸的驾驭方式
- 对威胁者的处理模式（拉拢/排除/利用）

---

### 🌿 辅助 / 情感型

**情感连接**：
- 与核心角色的情感纽带性质和强度
- 情感表达方式（主动/克制/替代行动）
- 牺牲的边界（会为谁牺牲什么）

**叙事功能**：
- 在哪些关键节点作为他人的情绪出口或转折催化剂
- 自身主线是什么（不仅仅是辅助他人）

---

## 输出要求

- 语言：中文
- 没有文本依据的维度：标注 `（原著文本不足，建议补充相关章节）`
- 有原文依据的结论：加引号并标注章节位置
- 输出结果直接用于生成 profile.md，要求具体可引用，不要写"可能""大概"这类模糊表述
